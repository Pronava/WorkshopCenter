## 🎧 MyAudioApp

- **技术栈**：JavaScript, Web Audio API, Vue.js, FastAPI  
- **简介**：一个音频可视化与比对系统，支持录音上传、音高检测与实时反馈。

- **核心功能：**
  - 音频切片上传与播放  
  - 实时人声音高与原曲比对反馈  
  - 波形图与音高走势的动态可视化
